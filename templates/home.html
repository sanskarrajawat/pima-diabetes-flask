About This Project

Diabetes Prediction is a Machine Learning based web application that predicts the risk of diabetes using medical parameters like Glucose, BMI, Blood Pressure and Age. The system is trained on the PIMA Diabetes dataset and provides instant results through a Flask web interface.

The goal of this project is to demonstrate how Artificial Intelligence can help in early health risk detection and support medical awareness.
